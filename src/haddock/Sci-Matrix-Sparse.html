sparse :: (Eq a, Num a) => (Int, Int) -> [(Int, Int)] -> [a] -> Either String (SparseMatrix a)

Function to create a sparse matrix from list of indices and list of values. Will fail if any index is out of bounds.

data SparseMatrix a

Data type for sparse matrices. This should be used instead of Matrix whenever possible. The bound assumes (0,0) in the top left corner, and values is a map between indices and nonzero values of the matrix.
