loadFile :: Read a => String -> String -> Char -> Int -> [Int] -> IO (Matrix a)

Read a file at a given path and load it into a matrix. Arguments: path = path to file comm = string indicating the beginning of a comment del = delimiter. If whitespace, then all whitespace will be treated as the delimiter. skip = number of rows to skip from the beginning of the file. This DOES include comment lines. cols = indices of columns to load, L to R, starting at 0. This is based on numpy.loadtxt.
