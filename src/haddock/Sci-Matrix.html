showMatrix :: Show a => Matrix a -> String

A variant of show that prints a matrix more neatly. Mostly used for debugging, but can be useful.

vectorFromList :: [a] -> Vector a

Create a vector from a list.

matrixFromLists :: [[a]] -> Either String (Matrix a)

Create a matrix from a list of list. Will only return a matrix if all of the lists are of the same length; otherwise returns a monadic failure. We take the lists to be the rows of the matrix.

linspace :: (Num a, Fractional a, Enum a, Eq a) => a -> a -> Int -> Either String (Vector a)

Create a column vector from the minimum value to the maximum value with n evenly spaced points. This is based on linspace from numpy. Note that linspace x x 1 will produce a monadic failure.

getColumn :: Int -> Matrix a -> Either String (Vector a)

Extract given column of a matrix (as a vector). Will produce a monadic failure if the desired column is out of range.

matrixToVector :: Matrix a -> Either String (Vector a)

Convert a matrix to a column vector. Will produce a monadic failure if the matrix is not a single column.

vectorToMatrix :: Vector a -> Matrix a

Convert a column vector to a matrix.
